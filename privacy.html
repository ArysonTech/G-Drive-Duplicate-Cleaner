Privacy Policy

Aryson Google Drive Duplicate Cleaner respects user privacy and is committed to protecting personal data.

Information Accessed

The application accesses Google Drive data only after the user explicitly signs in with Google and grants consent. The accessed data is limited to what is necessary to identify duplicate files.

Use of Data

Google Drive data is used solely for scanning, comparing, and managing duplicate files within the user's own account. Actions such as moving files to trash or permanent deletion are performed only after user confirmation.

Data Storage

The application does not store Google Drive data on external servers. All processing is performed securely for the user's account, and no data is shared with third parties.

Third-Party Sharing

No user data is sold, shared, or transferred to any third party.

User Control

Users can revoke Google account access to the application at any time through their Google Account permissions.
